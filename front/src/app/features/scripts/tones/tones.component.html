Tons et Demi-Tons

La plus petite distance qui sépare deux touches conjointes ( blanches ou noires ) se nomme :

Un demi-ton ( 1/2 ton ) !

Le demi-ton va ainsi devenir notre unité de mesure pour mesurer les distances entre les notes.

Si on compte le nombre de demi-tons entre deux notes distantes d'un octave.

( Par exemple, de Do3 à Do4 ).

Nous arrivons à un total de 12 demi-tons.

( ou 6 tons pour les cracks en mathématiques xD ).

Sur un clavier tempéré ( le standard actuel ), tous les demi-tons sont STRICTEMENT identiques.

Ce concept de mesurer les distances entre les notes est PRIMORDIAL pour comprendre ce qui va suivre !
