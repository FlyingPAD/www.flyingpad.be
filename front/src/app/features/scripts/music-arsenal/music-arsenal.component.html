Arsenal Musical

Il est vrai qu'il y a beaucoup de choses à apprendre pour devenir un bon musicien.

Il nous faut développer un véritable arsenal de connaissances et de techniques en tout genre.

Et bien que toutes ces connaissances vont devoir être exploitées simultanément.

Il est intéressant dans une optique d'apprentissage de les isoler pour mieux les travailler.

Je vous propose ainsi de concevoir votre arsenal musical en 3 disciplines distinctes :

▸ Arsenal Rythmique

( Gestion du Temps, Pulsation, Tempo, Valeurs de temps, Séquences, Rythmes, ... )

▸ Arsenal Harmonique

( Tonalités, Intervalles Harmoniques, Accords, Progressions Harmoniques, Modulations, ... )

▸ Arsenal Mélodique

( Intervalles Mélodiques, Gammes, Modes, Timbre, ... )

Avec de petites bases dans chacune de ces disciplines, il est déjà possible de faire pas mal de choses.

Nous allons ensuite combiner ces disciplines, ce qui nous ouvrira les portes de nouvelles disciplines.

( Improvisation, Composition, Arrangement, Production, ... )

À noter également que chacune de ces disciplines peut également devenir une spécialisation.

L'arsenal rythmique d'un percussioniste sera forcément plus développé que son arsenal harmonique.
