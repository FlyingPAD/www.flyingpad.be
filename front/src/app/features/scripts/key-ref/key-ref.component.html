La Référence Clavier

Pour illustrer les différents concepts théoriques,

je vais principalement me référer à un clavier de piano standard ( 88 touches ).

Pourquoi ?

Le clavier de piano offre de nombreux avantages :

  • Les notes sont positionnées dans l'ordre et en unique exemplaire.
  • Les notes naturelles sont colorées en blanc et les notes altérées en noir.
  • Il possède une tessiture très large (couvrant tous les instruments de l'orchestre).

Même si vous pratiquez déjà un autre instrument et/ou n'avez pas l'intention de devenir pianiste.

Le fait de visualiser ces concepts sur un clavier vous permettra fort probablement d'y voir plus clair.

  • De plus, le piano est très confortable à utiliser.

Inutile de s'abîmer les doigts sur des cordes ou à apprendre une technique de souffle particulière...

Ici, on pose simplement le doigt sur une touche et l'instrument produit un son relativement stable.

Pour toutes ces raisons,

Le piano est l'outil de "visualisation" par excellence.

Utilisé d'ailleurs dans tous les conservatoires, pour ces mêmes raisons.
