Les Intervalles

Un intervalle est la distance qui sépare deux notes.

Exemple :

La distance qui sépare les notes Do et Mi

( Tierce Majeure )

Dans le contexte musical, l'unité de mesure qui permet de calculer cette distance est le demi ton.

- La plus ' petite ' distance est donc 1/2 ton ( Seconde mineure )

- La plus ' grande ' distance est 12/2 tons ( Octave )

Il s'agit en fait, d'un rapport fréquentiel, d'un point de vue physique ou sonore peut se traduire en Hertz.

Dans la gamme tempérée, comme il existe 12 notes, il existe 12 intervalles.
