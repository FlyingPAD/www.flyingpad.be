Les Gammes

Une gamme, c'est une échelle d'intervalles.

Exemple :

Gamme diatonique majeure :

▸ M2, M3, P4, P5, M6, M7

On peut alors appliquer la formule à partir de n'importe quelle note.

Exemple :

En partant de Do :

▸ Do Ré Mi Fa Sol La Si

En partant de Ré :

▸ Ré Mi Fa# Sol La Si Do#

En partant de Mi :

▸ Mi Fa# Sol# La Si Do# Ré#

etc ...

Dans la création d'un contexte musical.

Les gammes définissent les notes que vous pouvez utiliser,

et par élimination, celles que vous devez éviter.

Chaque note de la gamme sera le point de départ d'un accord.

Quelques formules :

- Gamme pentatonique ( 5 notes )

▸ M2, M3, P5, M6

- Gamme par ton ( 6 notes )

▸ M2, M3, A4, A5, A6

- Gamme diatonique majeure ( 7 notes )

▸ M2, M3, P4, P5, M6, M7

- Gamme diatonique mineure ( 7 notes )

▸ M2, m3, P4, P5, m6, m7

- Gamme chromatique ( 12 notes )

▸ m2, M2, m3, M3, P4, Tr, P5, m6, M6, m7, M7
