La Notation Musicale

Parce qu'au bout d'un moment, il n'est plus possible de tout retenir par coeur.


Il sera donc nécessaire de déléguer une partie de sa mémoire à l'écrit.

Et inversément, écrire quelque chose permet de mieux le retenir ...

Connaître les différents systèmes de notation permet de trouver beaucoup plus de ressources.

Et ainsi, de progresser plus rapidement.


Sur notre planète, la langue internationale étant l'anglais.

Le système de notation anglo-saxon est naturellement devenu une référence.

- Du point de vue de la francophonie,

Les musiciens utilisent le système de notation européenne.

Nous utiliserons les notes Do, Ré, Mi, ...


- Du point de vue international,

Le système de notation anglo-saxon sera de rigueur.

Nous utiliserons alors les notes C, D, E, ...

Différentes représentations graphiques sont également à notre portée.

- Les partitions dites ' classiques '

- Les TAB's ( Pratique, pour les instruments à cordes )

- La notation en accords ( exemple : C C Bb F )
