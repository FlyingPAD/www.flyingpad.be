Les Gammes

Une Gamme, c'est une échelle d'intervalles.

Nous pouvons ainsi construire toutes sortes de gammes.

- Gamme à 5 notes ( Pentatonique )

- Gamme à 7 notes ( Diatonique )

- Gamme à 12 notes ( Chromatique )

Exemple :

L'échelle d'intervalles M2, M3, P4, P5, M6, M7 ( Gamme Diatonique Majeure )

Comme il est possible de démarrer à partir de n'importe quelle note ...

On peut considérer une gamme comme une formule.

Exemple :

Toujours en partant de l'échelle d'intervalles M2, M3, P4, P5, M6, M7 :

En partant de Do, nous aurons : Do Ré Mi Fa Sol La Si ( Do Majeur ).

En partant de Ré, nous aurons : Ré Mi Fa# Sol La Si Do# ( Ré Majeur ).

En partant de Mi, nous aurons : Mi Fa# Sol# La Si Do# Ré# ( Mi Majeur ).

etc ...
