Nommer les Notes

Exercice pratique :

Pour apprendre à les repérer du premier coup d'oeil.

1. Placez-vous face au clavier

2. Fermez les yeux

3. Tapez une note au hasard

4. Ouvrez les yeux

5. Nommez la note le plus vite possible

Exercice Bonus :

- Nommer la note dans les deux systèmes

- Chanter la note
